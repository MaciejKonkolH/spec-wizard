# AGENT BOOTLOADER: CryptoBot Financial Simulator (MVP)

## Tech Stack
- React + Vite + TailwindCSS (Optional for speed)
- Chart.js or Recharts
- Pure Client-Side Implementation (No backend/database)

## Core Logic Requirements
1.  **State Management**: Central store for ~15 parameters (sliders).
2.  **Yield Generator**:
    - Build a function that takes a `SEED` and returns an array of 24/60 monthly results.
    - Source data: `[3.5, 4.2, 5.1, -8.5, 2.9]` (representing real 5 months of trading).
    - Mode: Random selection from source for each month in sim.
3.  **Financial Cycle (Per Month)**:
    - `NewAUM = OldAUM * (1 + (MonthlyYield * Leverage))`
    - `If NewAUM > HWM: SuccessFee = (NewAUM - HWM) * SF_Rate; NewAUM -= SuccessFee; HWM = NewAUM`
    - `ManagementFee = NewAUM * (MF_Rate / 12); NewAUM -= ManagementFee`
    - `Apply Churn: NewAUM *= (1 - (BaseChurn * (Yield < 0 ? PanicMultiplier : 1)))`
    - `Apply Referrals: ReferralExpense = SuccessFee * Referral_Rate; CompanyProfit = SuccessFee - ReferralExpense`
    - `CompanyCashflow = CompanyProfit - FixedCosts - (MarketingBudget + (SuccessFee * MarketingReinvest))`
4.  **UI Layout**:
    - Sidebar for inputs (Inputs should be synced with number fields).
    - Grid for 4 Charts (Area for AUM, Bar for Revenue/Costs, Waterfall for Margin).
    - Big Number KPI Cards.

## Constraints
- **Zero Backend**: All persistence must be via URL serialisation (base64 of state in hash).
- **Performance**: Recalculate everything on every slider change (Main Thread is fine for 60 data points).
- **Design**: Premium Dark Mode, Glassmorphism components.

## Success Criteria
- Invester is impressed by visual feedback.
- Math is exact (+/- 0.01 USD).
- Total build time: < 48h for AI Agent.
