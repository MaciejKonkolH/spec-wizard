# Project Info: Model Finansowy CryptoBot

**Description:** Specyfikacja dynamicznego modelu finansowego dla projektu CryptoBot.

**Version:** 1.0

## Vision
Stworzenie interaktywnego symulatora (HTML/JS), który pozwoli na modelowanie przychodów (Success Fee), kosztów (Cloud/AI) oraz wzrostu kapitału (AUM) w czasie, uwzględniając specyfikę tradingu (High-Water Mark).
